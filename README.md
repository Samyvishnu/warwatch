## 🗺 What's Inside the Dashboard

- 43 conflict zones mapped (Critical / High / Medium / Monitor)
- Live ADS-B aircraft via OpenSky Network — auto refreshes every 15 seconds
- 6 switchable basemaps — Google Satellite, ESRI, Hybrid, Topo, Dark, Terrain
- GPS Jamming zones across 7 active regions
- Maritime AIS tracking — naval assets, tankers, dark vessels
- Satellite ground tracks — 16 orbital assets from 9 nations
- OSINT event stream with auto-updating alerts
- Maritime chokepoint status — Hormuz, Suez, Bab-el-Mandeb + more
- Search and fly-to any conflict zone instantly

## 🔧 Built With

- **Leaflet.js** — Map engine and layer management
- **OpenSky Network** — Free live ADS-B aircraft data
- **Google Maps Tiles** — Satellite and hybrid imagery
- **ESRI World Imagery** — Default satellite basemap
- **CelesTrak TLE** — Satellite orbital parameters
- **CartoDB Dark Matter** — Dark vector basemap
- **Vanilla HTML / CSS / JS** — Zero frameworks, zero install
- **Claude AI** — Built collaboratively with AI assistance
