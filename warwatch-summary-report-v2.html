01 PLATFORM OVERVIEW WHAT IS WARWATCH

WARWATCH is a fully self-contained, browser-based geospatial intelligence dashboard that aggregates, visualizes, and monitors global conflict data in real time. Every data source is publicly accessible — no classified feeds, no paid APIs, no authentication required. The platform runs as a single index.html file — open it in any modern browser and the full intelligence picture loads instantly.

Version 2 adds two major new dedicated tracker panels — a Live Flight Tracker and a Live Fleet Tracker — each with full filterable data tables, emergency alert systems, click-to-navigate functionality, and real-time data synced with the map.

✓ ZERO BACKEND — runs entirely in-browser as a single HTML file
✓ ZERO API KEYS — all data sources are open and free
✓ ZERO INSTALL — open in Chrome, Firefox, Edge, or Safari
✓ CLEAN URL — renamed to index.html for shortest possible shareable link

SQUAWK EMERGENCY DETECTION SYSTEM
